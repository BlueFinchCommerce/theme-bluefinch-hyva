# BlueFinch Hyvä Theme

## Overview
The BlueFinch Hyvä Theme is a Magento child theme based on the Hyvä framework. Hyvä is a modern, lightweight, and high-performance frontend solution for Magento 2, designed to replace the traditional themes with a more efficient and developer-friendly alternative. By using Alpine.js and Tailwind CSS, Hyvä significantly improves page speed, user experience, and maintainability.

This theme is designed to work seamlessly with the BlueFinch Build module, allowing store administrators to customize basic styles such as logos, colors, and fonts using CSS variables, ensuring a consistent and branded storefront without the need for direct theme file modifications.

## Features
- Built on the Hyvä framework for optimal performance
- Fully compatible with the BlueFinch Build module for easy customization
- Configure store branding through the Magento admin panel

## Configuration
1. Log in to the Magento Admin Panel.
2. Navigate to **Content > Design > Configuration**.
3. Select the store view you want to customize and apply the BlueFinch Hyvä Theme.
4. Use the BlueFinch Build module under **Stores > Configuration > BlueFinch > Theme** to modify logos, colors, and fonts.
5. Save changes and refresh the cache if necessary.

## Usage
Once configured, the BlueFinch Hyvä Theme will apply the selected branding options dynamically, ensuring a modern and optimized storefront experience.
